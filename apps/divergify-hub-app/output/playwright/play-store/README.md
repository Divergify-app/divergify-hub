# Divergify Play Store Assets

This folder contains the current Play submission package pieces generated from the local app build and approved constellation branding.

## Contents

- `phone/`
  - Play-sized phone screenshots at `1080 x 1920`
- `raw/`
  - viewport captures used to generate the phone screenshots
- `branding/`
  - `hi-res-icon-512.png`
  - `feature-graphic-1024x500.png`

## Generation

- Phone screenshots:
  - `scripts/capture_play_store_screenshots.sh`
- Branding graphics:
  - `scripts/build_play_store_graphics.py`

## Notes

- The screenshot set is submission-shaped.
- The icon and feature graphic are generated from the current approved constellation mark and light wordmark.
- If branding changes again, regenerate the graphics before upload.
